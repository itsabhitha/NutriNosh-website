## Features
- Track daily calories, weight, and water intake
- Visual weekly summaries with charts
- Light & dark mode support
- Responsive design for mobile and desktop

## Tech Stack
- HTML, CSS, JavaScript
- Tailwind CSS for styling
- Chart.js (for charts)
